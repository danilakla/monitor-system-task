Для запуска клиентской части приложения, следуйте следующим шагам:

1. Установите Node.js, если у вас его еще нет на компьютере.

2. Откройте командную строку или терминал и выполните следующую команду:
	npm install -g @angular/cli@14.0.0

3. Перейдите в корневую папку проекта клиента.

4. Запустите следующую команду: ng serve

Эта команда запустит клиентскую часть приложения и будет слушать на порту по умолчанию (4200). Вы должны увидеть сообщение о том, что сервер успешно запущен.

----------------------------------------------------------
Для запуска серверной части и базы данных PostgreSQL с использованием Docker, следуйте следующим шагам:

1. Перейдите в корневую папку c docker-compose.yml (\monitor-system-task\monitorservice\docker-compose.yml)	
2. Запустите следующую команду: docker-compose up
3. Приложения будет слушать порт по умолчанию (8080)

----------------------------------------------------------
Если у вас нет возможности использовать Docker, следуйте следующим шагам:

1. Установите PostgreSQL последней версии на ваш компьютер.
2. Создайте пользователя с именем "admin" и паролем "123" в PostgreSQL.
3. Создайте базу данных с названием "monitor" в PostgreSQL.
4. Убедитесь, что на вашем компьютере установлена Java SDK версии 11 (например, Corretto-11).
5. Запуск сервера, будет слушать порт по умолчанию (8080)
